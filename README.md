# AI Problem Solving Assignment

## 📌 Problems Implemented

### 1. Insurance Claim Decision System
A rule-based system using propositional logic to evaluate whether an insurance claim should be approved or rejected based on input conditions.

### 2. Student Performance Predictor
A simple regression-based system that predicts student performance based on study hours and attendance.

---

## ⚙️ Algorithms Used
- Propositional Logic (Rule-Based Inference)
- Regression (Mathematical Prediction)

---

## ▶️ Execution Steps

### Insurance System
1. Navigate to insurance_system folder  
2. Run: python main.py  
3. Enter required inputs  
4. View decision  

### Student Predictor
1. Navigate to student_predictor folder  
2. Run: python main.py  
3. Enter study hours and attendance  
4. View predicted score
